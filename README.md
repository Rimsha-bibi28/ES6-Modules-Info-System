# ES6 Modules Info System

Professional ES6 Modules project demonstrating the separation of teacher and student information with dynamic rendering on a clean, animated, and responsive UI.

## Features
- ES6 Modules for modular code structure
- Dynamic display of teacher and student information
- Elegant, responsive, and animated CSS interface
- Minimal, professional, and user-friendly design

Open index.html in a modern browser. For full ES6 module support, use a local server like VS Code Live Server.

Usage

The teacher and student information is displayed dynamically on page load.

Hover over cards to see smooth lift animations.
